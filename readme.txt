== Description ==

Google +1 (Social Button) is a social recommend plugin that let users recommend or +1 your content.
The Google +1 plugin enable users to see how many users already +1 this page, and which of their friends recommend it too.
The +1 button will influence your website's search ranking.
Add Google +1 to your pages to help your site stand out.

= Features =

* Easy installation and customization
* Support for All 44 languages of Google +1 API Locales
* Support live count
* Quick loading time
* Settings allows for the +1 button to be placed on pages, posts,or home page

== Installation ==

Follow the steps below to install the plugin.

1. Upload the 'google-plusone-social-button' directory to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Settings'->'Google +1 Button' to configure the google+1 button.
